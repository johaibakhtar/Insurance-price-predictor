🏥 Insurance Price Prediction

A Machine Learning project that predicts medical insurance charges based on personal and health-related attributes using Python and Linear Regression.

🚀 Project Overview

Healthcare costs vary significantly depending on multiple factors such as age, BMI, smoking habits, and region.
The goal of this project is to analyze the dataset and build a predictive model that estimates insurance charges.

This project demonstrates the complete data science pipeline:

Data Analysis

Data Visualization

Feature Engineering

Machine Learning Modeling

Model Evaluation

📊 Dataset

The dataset contains medical insurance information for individuals.

Feature	Description
age	Age of the individual
sex	Gender
bmi	Body Mass Index
children	Number of children covered
smoker	Smoking status
region	Residential region
charges	Medical insurance charges
